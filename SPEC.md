# Project Specification

## Context
This project defines a spec-driven AI-assisted coding workflow.
The specification is the execution contract.

## Scope
Included:
- Task-defined feature work

Excluded:
- Architecture changes
- Dependency changes
- CI/CD modifications

## Constraints
Always:
- Run tests
- Follow naming conventions

Ask First:
- Schema changes
- API changes

Never:
- Commit secrets
- Modify pipelines
